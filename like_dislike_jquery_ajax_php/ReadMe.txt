Instructions::

1. Move "like_dislike_jquery_ajax_php/" folder at your localhost.
2. Create a database, like "codexworld".
3. Open the "tutorial.php" file => change the database host, database username, database password and database name into the __construct() function.
4. Import the "tutorials.sql" file into the database.
5. Now run "http://localhost/like_dislike_jquery_ajax_php/" URL at the browser.
6. Test like and dislike system.
