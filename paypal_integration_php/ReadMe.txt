============ Instruction ============

1. Create a database (codexworld) and import the products_payments.sql file into the database (codexworld).

2. Open the "db_config.php" file and change $dbHost, $dbUsername, $dbPassword and $dbName variable values with your database credentials.

3. Open the "products.php" file and change the $paypal_id variable value with your PayPal business email.

4. Now run the "products.php" file and test the functionality.
